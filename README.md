### mosch
memo save & search, 备忘录 保存和搜索小项目. 自己运行记录一些事情

```bash
# 进入到 mosch 项目目录
cd mosch

# 引入包依赖
make vendor

# 编译项目
make build

# 运行项目
make run
```
